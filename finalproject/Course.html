Storing Integers in Two's Complement

Two's Complement is a way of representing integers in bit patterns.
In Short, each bit represents 2^n. Where 'n' is dependent on the position of the bit starting from the right hand side.
The leftest most bit is called a sign bit which shows whether the integer stored is a negative number (bit is equal to 1) or if its a positive number (bit is equal to 0)
If a is 1, then that bit is equal to 2^n, but if its a 0 then that bit is equal to 0. To find the integer stored, you must find the sum of all the bits.
